## Features

- 目前支持的功能:
    - 游客评论、github 评论
    - 支持 `markdown` 语法
    - 支持 `emoji` 表情，留言区输入符号 `:` 自动补全表情 🍭
    - 文章编辑支持图片拖拽上传
    - 文章主题支持语法高亮
    - 支持 ElasticSearch 搜索 🔍
    - 支持搜索结果高亮显示
    - 评论实时同步
    - 新文章广播通知
    - 文章置顶和隐藏
