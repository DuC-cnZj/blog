# duc's blog
----------
一个花了一周左右时间完成的博客项目。

## Required
- docker-compose

## 安装
--------
- `$ git clone 本项目` 
- `$ cd run` 进入到 run 目录下面 :smile:
- `$ docker-compose up app`

done 🐳 👏

## 预览
- 博客主页 `http://localhost:8003`
- 博客后台 `http://localhost:8002`
- api `http://localhost:8001`
